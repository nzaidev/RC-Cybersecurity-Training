SOC 2 Compliance

What is SOC 2?

SOC 2 is like a report card that shows our clients we're serious about protecting their information. It's like a restaurant getting a health department rating.

The Five SOC 2 Trust Principles

🔒 Security

Do we protect information from unauthorized people? (Like having locks on doors)

⚡ Availability

Are our systems working when clients need them? (Like keeping the lights on)

✅ Processing Integrity

Do we handle information accurately? (Like a bank making sure transactions are correct)

🤐 Confidentiality

Do we keep secrets secret? (Like a doctor keeping medical records private)

🛡️ Privacy

Do we handle personal information properly? (Like asking permission before sharing someone's phone number)

Your Role in SOC 2

SOC 2 isn't just an IT responsibility - it requires everyone to follow security practices. When auditors check our compliance, they'll look at how well all employees follow security procedures.

Password Security

Passwords Are Like Keys to Your House

Just as you wouldn't want your house key to be easy to copy or guess, your passwords should be hard for criminals to figure out.

❌ Weak Passwords

  • "password123" - Common word + predictable numbers
  • "Roig2024" - Company name + current year
  • "admin" - Dictionary word
  • "P@ssw0rd" - Simple substitutions

✅ Strong Passwords

  • "Tr@il$Hiking&Coffee2024!" - Complex, long, unpredictable
  • 12+ characters minimum
  • Mix of uppercase, lowercase, numbers, symbols
  • Unique for each account

Creating Strong Passwords: The Sentence Method

  1. Think of a memorable sentence: "I love hiking trails with my dog every Saturday morning"
  2. Take first letters: "Ilhtwmdesm"
  3. Add numbers and symbols: "Ilhtwmdesm2024!"
  4. Mix in uppercase: "IlhtWmdesM2024!"

Multi-Factor Authentication (MFA)

MFA is Like Having Multiple Locks on Your Door

Even if someone guesses your password (picks your first lock), they still can't get in because they need a second form of identification.

Three Types of "Proof" You Can Provide

🧠 Something You KNOW

  • Password
  • PIN
  • Security question answer

📱 Something You HAVE

  • Phone
  • Security token
  • Smart card
  • Email access

👤 Something You ARE

  • Fingerprint
  • Face recognition
  • Voice recognition

🏦 Real-World Example

Banking ATM: You need your card (something you have) AND your PIN (something you know)

Best Practice: Use Authenticator Apps

Google Authenticator, Microsoft Authenticator, or Authy are more secure than SMS text messages and work without internet connection.

Recognizing Phishing Attacks

What is Phishing?

Phishing is like fishing, but instead of using bait to catch fish, criminals use fake emails, websites, or messages to "catch" your personal information, passwords, or money.

Red Flags to Watch For

⚠️ Language Red Flags

  • Urgent language: "Immediate action required"
  • Threatening: "Your account will be closed"
  • Generic greetings: "Dear Customer"
  • Emotional manipulation: "You've won!"

🔍 Technical Red Flags

  • Suspicious sender addresses
  • Mismatched URLs (hover to check)
  • Unexpected attachments
  • Poor grammar/spelling

The "Hover Test" - Important Skill!

  1. Receive an email with a link
  2. Don't click immediately - pause and be suspicious
  3. Hover your mouse over the link without clicking
  4. Look at the actual URL that appears
  5. Compare it to what you expect

How to Respond to Suspicious Messages

The STOP-THINK-VERIFY-REPORT Process

1. STOP 🛑

  • Don't click any links
  • Don't download attachments
  • Don't provide information
  • Don't panic or rush

2. THINK 🤔

  • Is this message expected?
  • Does the request make sense?
  • Is the sender who they claim to be?
  • What would happen if this is fake?

3. VERIFY ✅

  • Contact sender through different method
  • Check official website directly
  • Ask supervisor or IT if unsure
  • Look up sender's phone number independently

4. REPORT 📞

  • Forward suspicious emails to: security@roigcommunications.com
  • Report suspicious calls to supervisor
  • Document what happened and when
  • Don't be embarrassed - reporting helps everyone

Device Security & Safe Browsing

Your Antivirus = Your Computer's Immune System

Antivirus software recognizes harmful software and stops it before it can damage your computer or steal information.

✅ DO

  • Keep antivirus running always
  • Allow automatic updates
  • Run regular scans
  • Pay attention to alerts
  • Report problems to IT

❌ DON'T

  • Disable protection to install software
  • Ignore warnings
  • Install multiple antivirus programs
  • Click "Allow" unless absolutely sure

Safe Browsing Guidelines

Dangerous Download Sources to Avoid

  • Peer-to-peer networks (BitTorrent, LimeWire) - Often contain malware
  • Suspicious software sites - "Free" versions of expensive software
  • Pop-up advertisements - "Download now" buttons are often malware
  • Email attachments from strangers - Never open these

Software Updates: Your Digital Armor

Think of Updates Like Fixing Holes in a Fence

Criminals constantly look for vulnerabilities (holes) in software, and updates patch these holes.

Update Priority Levels

🚨 Critical Security Updates (Install Immediately)

  • Operating system patches (Windows Updates, macOS updates)
  • Browser security updates (Chrome, Firefox, Safari, Edge)
  • Security software updates (antivirus, firewall)

Why urgent: These fix vulnerabilities that criminals are actively exploiting

⚠️ Important Software Updates (Install Within a Week)

  • Business applications (Microsoft Office, Adobe products)
  • Communication tools (email clients, messaging apps)

✅ Company Update Policy

  • Enable automatic updates for operating systems and browsers
  • Install critical security patches within 48 hours
  • Allow restarts when needed - many updates require a computer restart

Personally Identifiable Information (PII)

What is PII?

PII is any information that can be used to identify a specific person. Think of it like puzzle pieces - combine several pieces and you can identify the person.

Types of PII (Most to Least Sensitive)

Direct Identifiers

Can identify someone by themselves

  • Social Security Numbers
  • Driver's License Numbers
  • Credit Card Numbers
  • Bank Account Numbers

Indirect Identifiers

Can identify when combined

  • Full Name + Date of Birth
  • Address + Phone Number
  • Email + Employer

PII Handling Steps

Collection

Only take what you need for your job

Storage

Use encrypted systems, limit access

Usage

Only use for intended purpose

Sharing

Verify need, get authorization, use secure methods

Disposal

Shred physical documents, use secure deletion

Remote Work Security

VPN: Your Secure Tunnel

A VPN is like having a private, encrypted tunnel between your computer and the company's network. Even in a coffee shop, all your traffic goes through this secure tunnel.

When You MUST Use VPN

  • 🏢 Accessing any company systems (email, file servers, business applications)
  • ☕ Working on public Wi-Fi (coffee shops, airports, hotels, libraries)
  • 🤝 Connecting to client systems
  • 🏠 Working from home (for consistency and best practice)

✅ VPN Best Practices

  • Connect to VPN first, before accessing company resources
  • Verify connection status - look for VPN icon
  • Keep VPN running throughout work session
  • Contact IT immediately if problems occur

⚠️ Wi-Fi Network Safety

  • Safest: Your secure home network (WPA3/WPA2)
  • Okay: Company guest network
  • Risky: Public Wi-Fi (use VPN required)
  • Never: Open/unsecured networks

Security Incident Response

What is a Security Incident?

Any event that threatens the safety of our computer systems, data, or business operations. Think of it like a medical emergency - the faster you get help, the better the outcome.

Warning Signs to Watch For

🖥️ Computer Issues

  • Unexpected slowdowns or crashes
  • Strange network activity
  • Unusual applications running
  • Files you didn't create

👤 Account Issues

  • Unexpected password changes
  • Login notifications from unknown locations
  • Emails you didn't send in "Sent" folder
  • Friends saying they received strange messages from you

📁 File Issues

  • Missing files
  • Files with strange extensions (.encrypted, .locked)
  • Desktop wallpaper changed to ransom message
  • Unable to open files that previously worked

🚨 If You See Ransomware

DON'T try to recover files yourself
DO disconnect and report immediately

The STOP-PRESERVE-REPORT Process

1. STOP 🛑

  • Don't continue using affected system
  • Don't try to "investigate"
  • Don't try to "fix it"
  • Don't shut down unless told to

2. PRESERVE 📸

  • Don't delete anything
  • Take photos of error messages
  • Write down what happened
  • Note the time and what you were doing

3. REPORT 📞

Contact Information:

  • Emergency Security: [Phone] - 24/7
  • IT Security Email: security@roigcommunications.com
  • IT Manager: [Phone and email]

Information to Provide:

  • What happened and when
  • Which systems are affected
  • What you were doing when it occurred
  • Current status
  • Your contact information

Remember: You're Not Expected to Be a Security Expert

Your job is to recognize when something might be wrong and get help. When in doubt, report it. False alarms are much better than missed real incidents.
